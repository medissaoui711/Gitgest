# 🤖 GitGest Privacy Policy

Last updated: May 2026

## TL;DR

GitGest collects zero personal data. Everything runs locally in your browser.

---

## 1. What We Collect

**Nothing.** GitGest does not collect, store, transmit, or share any personal data, usage data, or browsing history.

## 2. How GitGest Works

GitGest operates entirely within your browser:

- When you click the AI Context button, your browser makes direct API calls to `api.github.com` — the same requests your browser makes when you visit GitHub normally.
- The extracted context is written directly to your clipboard or downloaded as a file.
- No data passes through any GitGest server — because there are no GitGest servers.

## 3. GitHub Personal Access Token (PAT)

If you choose to enter a GitHub PAT for private repo access:

- The token is stored locally using `chrome.storage.sync` (Chrome's built-in encrypted storage).
- It is never transmitted to any server other than `api.github.com` directly from your browser.
- You can delete it at any time from the GitGest popup settings.

## 4. Local Storage

GitGest uses `chrome.storage.sync` to save your preferences:

- Output format preference (Markdown or JSON)
- Daily export count (for free tier tracking)
- GitHub PAT (if provided)

This data never leaves your device except through Chrome's standard sync mechanism if you have Chrome Sync enabled.

## 5. Third-Party Services

GitGest makes requests to **api.github.com** only — GitHub's official REST API. These requests are subject to [GitHub's Privacy Policy](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

GitGest has no affiliation with GitHub, Inc.

## 6. Analytics

GitGest contains **zero analytics, tracking pixels, or telemetry** of any kind. We do not use Google Analytics, Mixpanel, or any similar service.

## 7. Children's Privacy

GitGest does not knowingly collect any information from children under 13. The extension is a developer tool intended for adults.

## 8. Changes to This Policy

If we ever update this policy, the new version will be published at this URL with an updated date. Significant changes will be noted in the Chrome Store listing.
