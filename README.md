> **One click. Any GitHub repo. AI-ready context.**
> Stop manually copying files. GitGest extracts repo structure, tech stack, and commits into a formatted prompt in 3 seconds.

---

## The Problem

Opening a GitHub repo in ChatGPT or Claude requires:

1. Manually reading the file tree
2. Copying `package.json` / `requirements.txt`
3. Noting the repo structure
4. Writing a context prompt

**That's 10 minutes of work. GitGest makes it 3 seconds.**

## Installation

### Chrome Web Store (Recommended)

[Install from Chrome Store →](https://chromewebstore.google.com) *(pending review)*

### Developer Mode (from source)

```bash
git clone https://github.com/your-handle/gitgest
cd gitgest
npm install
npm run build
```

1. Open `chrome://extensions`
2. Enable **Developer mode** (top right)
3. Click **Load unpacked** → select `dist/` folder

## Tech Stack

- **Manifest V3** — modern Chrome extension standard
- **esbuild** — <50KB bundle (zero dependencies)
- **GitHub REST API v3** — 3 parallel calls via `Promise.all()`
- **100% local** — no servers, no tracking, no data collection

---

## Architecture

```
content.js (entry)
    │
    ├── observer.js         SPA navigation (GitHub turbo-frame)
    ├── extractor.js        DOM parse + API orchestration
    │   ├── api-client.js   3 parallel GitHub API calls
    │   └── tech-detector.js  package.json / requirements.txt parsing
    └── output/
        ├── markdown.js     Formatted markdown builder
        └── clipboard.js    Clipboard write + fallback modal
```
