## Long Description (EN)

**GitGest — GitHub Repo to AI Context in 3 Seconds**

Stop wasting 10 minutes manually preparing GitHub repos for AI tools.
GitGest adds a single button to every GitHub repo page — click it and get a perfectly structured AI prompt in your clipboard instantly.

**How it works:**

1. Open any GitHub repository
2. Click the "🤖 AI Context" button (next to the Fork button)
3. Paste into ChatGPT, Claude, Gemini, or any AI tool

**What's included in the context:**
• Repository metadata (stars, forks, language, topics, license)
• Detected tech stack (auto-parsed from package.json, requirements.txt, go.mod, Cargo.toml, and more)
• Smart file tree — source code files only, binary files excluded
• Key source files ranked by importance
• Last 5 commits with author and relative date
• Output in Markdown or JSON format

**Why GitGest?**
✓ Works on any public GitHub repository — no setup required
✓ Private repos supported with your GitHub Personal Access Token
✓ 100% local — no servers, no data collection, no accounts needed
✓ Tiny footprint: 26KB bundle, injects in <100ms
✓ Survives GitHub's SPA navigation — stays active as you browse
✓ Smart tech detection: groups related packages (Jest 29.4 (8 pkgs) instead of 8 separate lines)

**Free Tier:**
• 5 exports per day
• Public repositories
• Markdown output

**Pro (with GitHub PAT — free):**
• Unlimited exports
• Private repositories
• JSON output format
• 5,000 API requests/hour

**Perfect for:**
→ AI Prompt Engineers preparing repo context for LLMs
→ Code reviewers getting quick repo overviews
→ Tech leads onboarding to new codebases
→ Developers using ChatGPT, Claude, Copilot, or Cursor

**Privacy:**
GitGest processes everything locally in your browser. No data is sent to any server. GitHub API calls go directly from your browser to api.github.com. We collect zero telemetry.
